Visualizing New York City Weather: 1876 to 2014

This visualization, created in D3.js using NOAA weather data, is an updated, interactive version of this print-based classic visualization of 2003 NYC weather data by the New York Times.

The visualization shows temperature and precipitation data collected by the New York Central Park Belvedere Castle National Weather Service Observatory, obtained from NOAA's FTP service. Any year back to the start of collection in 1876 can be visualized.

Full source code for the visualization (including a Python script to download, clean and reformat the data and calculate the various metrics) can be found on GitHub.



Slide the slider or click the year forward/back buttons to select a different year to display, and mouseover the visualization to read temperature max/min and cumulative preciptation details.
